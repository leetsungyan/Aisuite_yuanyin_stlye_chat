## Features

- Transform any unfortunate event into a super positive post
- Uses AI to reframe situations with Taiwanese-style optimism
- Web interface for easy interaction
- Supports multiple AI providers (Groq, OpenAI, etc.)

## Setup

1. Clone this repository
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Set up your API keys (see below)
4. Run the app:
   ```bash
   streamlit run app.py
   ```

## API Keys

You need API keys for AI providers. Set them as environment variables:

- `GROQ_API_KEY`: For Groq AI
- `OPENAI_API_KEY`: For OpenAI (optional)
- `MISTRAL_API_KEY`: For Mistral (optional)

## Deployment

This app can be deployed on:

- Any platform supporting Python web apps
